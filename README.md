# Exploratory Data Analysis (EDA) - Student Dataset

## 📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of a student dataset. The main goal is to understand the structure of the data, identify patterns, detect anomalies, and extract meaningful insights using statistical analysis and visualization techniques.

---

## 📊 Objectives
- To understand the dataset structure (rows, columns, data types)
- To find missing values and data inconsistencies
- To analyze relationships between variables
- To identify trends and patterns in student performance
- To detect outliers and anomalies
- To summarize key insights from the data

---

## 📁 Dataset Information
The dataset contains student-related information such as:
- Gender
- Age
- Subjects
- Marks/Grades

---
## 📌 Conclusion
EDA helped in understanding the dataset clearly and revealed important patterns related to student performance. These insights can be useful for improving academic strategies and decision-making.
